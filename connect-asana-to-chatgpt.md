# Connect Asana to ChatGPT

Query your Asana projects and tasks directly from ChatGPT using CorpusIQ.

## Setup

1. [Connect Asana](https://corpusiq.io) in your CorpusIQ dashboard
2. Add CorpusIQ to ChatGPT as an MCP server: `https://mcp2.corpusiq.io/mcp`
3. Start asking questions

## Example Questions

- "What tasks are due this week across all projects?"
- "Show me all unassigned tasks in the Marketing project"
- "Which projects are behind schedule?"
- "Summarize completed tasks from last sprint"

## What You Can Access

- Projects, tasks, and subtasks
- Assignments and due dates
- Custom fields and tags
- Comments and attachments

## Security

CorpusIQ uses read-only access. ChatGPT can query your Asana data but cannot create, update, or delete tasks.

See all [connectors](../connectors.md) for other tools you can connect.
