# Quickstart

Get up and running in 30 minutes. Five steps from signup to your first real question.

## The path

| Step | What | Time |
|------|------|------|
| [1. Create account](01-create-account.md) | Sign up at corpusiq.io | 2 min |
| [2. Connect Claude](02-connect-claude.md) | Plug CorpusIQ into Claude Desktop | 3 min |
| [3. Connect ChatGPT](03-connect-chatgpt.md) | Or plug into a ChatGPT custom GPT | 3 min |
| [4. Authenticate first connector](04-first-connector.md) | Connect QuickBooks, Shopify, GA4, etc. | 2 min |
| [5. Fire first prompt](05-first-prompt.md) | Ask a question, get real data back | 2 min |
| [6. Your first cross-source query](06-first-cross-source-query.md) | Connect two tools, ask something that needs both | 5 min |

**Total: ~30 minutes** (or 25 if you skip the ChatGPT step).

## Start here

New to CorpusIQ? Start at [step 1](01-create-account.md).

Already signed up? Jump to [step 4](04-first-connector.md) (authenticate a connector).

Already have a connector authenticated? Try [step 6](06-first-cross-source-query.md) (multi-connector queries).
