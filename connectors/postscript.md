# Postscript

## What it unlocks
Bring your Postscript SMS marketing data into conversation with the rest of your stack. Ask about subscriber growth, opt-in keywords, and how SMS compares to your email and ad channels.

## Before you connect
- A Postscript account connected to a Shopify store.
- A Postscript API key with read access. Generate one in Postscript at Settings → API.
- About 2 minutes.

## How to connect
1. In Postscript, go to Settings → API and create a new API key (or copy an existing one).
<!-- screenshot: Postscript API key settings page -->
2. Open your CorpusIQ dashboard and click Connections.
3. Find Postscript and click Connect.
4. Paste the API key into the field and save.
<!-- screenshot: CorpusIQ Postscript API key form -->

You'll see Postscript change from gray to green in your CorpusIQ dashboard.

## What CorpusIQ can see
- Your SMS subscriber list with phone numbers and Shopify customer linkage.
- Subscriber details: opt-in date, source, status.
- Your active opt-in keywords.

Read-only. CorpusIQ never sends SMS, edits subscribers, or changes keywords.

## Questions you can ask
- "How many new SMS subscribers did we get last month?"
- "Show me my active Postscript keywords."
- "Which Postscript subscribers are also Shopify customers?"
- "Find SMS opt-ins from the last 7 days."

## Troubleshooting
- "Invalid API key" — regenerate the key in Postscript and paste it again. Whitespace around the key will break it.
- No subscribers showing — confirm the API key was issued for the correct Postscript shop.
- Missing fields — Postscript's API doesn't expose individual message-level metrics here; use Postscript's own dashboard for campaign analytics.
