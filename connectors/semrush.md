# Semrush

## What it unlocks
Bring SEO and competitive search data into the conversation. Ask which keywords your competitors rank for, what your backlink profile looks like, and where you're losing organic ground — without leaving the chat.

## Before you connect
- A Semrush account with API access. The Business plan or higher is required for the API.
- Your Semrush API key (Settings → API → API units).
- About 2 minutes.

## How to connect
1. In Semrush, navigate to your profile → Subscription info → API units, and copy your API key.
<!-- screenshot: Semrush API key page -->
2. Open your CorpusIQ dashboard and click Connections.
3. Find Semrush and click Connect.
4. Paste the API key and save.
<!-- screenshot: CorpusIQ Semrush API key form -->

You'll see Semrush change from gray to green in your CorpusIQ dashboard.

## What CorpusIQ can see
- Domain overview: organic and paid traffic.
- Top organic keywords any domain ranks for.
- Top paid (Google Ads) keywords any domain bids on.
- Organic competitor list.
- Backlink overview: authority, referring domains.
- Keyword research: search volume, CPC, difficulty.
- Domain history (month-by-month snapshots).

Read-only. CorpusIQ never spends Semrush units beyond the queries you ask for.

## Questions you can ask
- "What organic keywords does competitor.com rank for?"
- "Show me my domain's SEO overview."
- "How does my backlink profile compare to competitor.com?"
- "What's the search volume and CPC for 'enterprise CRM'?"
- "Who are my top organic competitors?"

## Troubleshooting
- "API key invalid" — confirm your Semrush plan includes API access (Business or higher) and that you have remaining API units.
- "No data for domain" — Semrush data lags reality; very new domains may not be indexed yet.
- High API unit consumption — each query costs units. Ask narrower questions to control spend.
