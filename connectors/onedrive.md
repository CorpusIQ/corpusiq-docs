# OneDrive

## What it unlocks
Let CorpusIQ read your OneDrive files when answering questions — contracts, spreadsheets, board decks, prior reports. Stop digging through folders to find the doc you wrote three months ago.

## Before you connect
- A Microsoft 365 work or personal account.
- About 1 minute. If you already connected Outlook, OneDrive may already be authorized under the same Microsoft connection.

## How to connect
1. Open your CorpusIQ dashboard and click Connections.
2. Find OneDrive (or Microsoft 365) and click Connect.
<!-- screenshot: Microsoft connection card in CorpusIQ -->
3. Sign in with your Microsoft account.
4. Review the permissions and click Accept.
<!-- screenshot: Microsoft consent screen listing OneDrive scope -->
5. You'll return to CorpusIQ.

You'll see OneDrive change from gray to green in your CorpusIQ dashboard.

## What CorpusIQ can see
- Your OneDrive file list and folder structure.
- File contents you ask it to read: Word, Excel, PowerPoint, PDFs, text files.
- Storage usage, recent files, and files shared with you.

Read-only. CorpusIQ never edits, deletes, or shares your files.

## Questions you can ask
- "Find my OneDrive file about the Q3 board meeting."
- "What does the latest pricing spreadsheet say?"
- "Show me files I opened recently."
- "Search OneDrive for any contract with Acme Corp."

## Troubleshooting
- "Consent required" — your Microsoft 365 admin restricts third-party apps. Ask them to approve CorpusIQ for your tenant.
- File not found — confirm the file lives in OneDrive (not SharePoint or Teams shared libraries).
- Permission denied on a specific file — that file was shared with restricted access. Open it in OneDrive to confirm.
