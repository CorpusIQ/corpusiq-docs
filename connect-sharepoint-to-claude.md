# Connect SharePoint to Claude

Query your SharePoint documents and lists directly from Claude using CorpusIQ.

## Setup

1. [Connect SharePoint](https://corpusiq.io) in your CorpusIQ dashboard
2. Add CorpusIQ to Claude Desktop as an MCP server: `https://mcp2.corpusiq.io/mcp`
3. Start asking questions

## Example Questions

- "Find all documents related to Q3 budget"
- "What's in the onboarding checklist?"
- "Show me the latest version of the sales playbook"
- "Search for policies that mention data retention"

## What You Can Access

- Documents and file metadata
- Lists and libraries
- Site pages and content
- Search across all sites

## Security

CorpusIQ uses read-only access. Claude can query your SharePoint content but cannot upload, modify, or delete documents.

See the [Claude connector guide](connectors/ai-mcp-server-for-business-data.md) for detailed setup.
