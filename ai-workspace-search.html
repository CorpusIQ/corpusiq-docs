    AI Workspace Search — One Question Across Every Tool¶

    Stop switching dashboards. AI Workspace Search lets you ask a single question and get answers pulled simultaneously from Stripe, Shopify, QuickBooks, and every other tool your business runs on.

    What Is AI Workspace Search?¶

    AI Workspace Search is the query layer that bridges AI Workspace ↔ Business Intelligence. It transforms the way teams interact with business data — instead of logging into each platform, navigating to the right report, and piecing together answers manually, you ask one natural-language question and get a single, sourced answer.

    Ask: "What was our total revenue last quarter, broken down by channel?"

    Get back: A unified answer with data from Stripe (subscription payments), Shopify (online store sales), and QuickBooks (wholesale invoices) — every figure linked to its source.

    Cross-Tool Search: One Question, Every Answer¶

    Ask Once. Search Everywhere.¶

    AI Workspace Search connects to your entire stack and queries every connected tool in parallel. No more context-switching between dashboards.

    QuickBooks P&L, expenses, accounts receivable, tax data

    Snowflake / BigQuery Custom SQL queries, warehouse data, company-wide KPIs

    Every connector responds in seconds. The AI correlates results across tools and surfaces the full picture — not just the first hit.

    Source Citations for Every Answer¶

    Trust is built on traceability. Every answer from AI Workspace Search includes source citations — so you see exactly which tool provided each data point, when it was queried, and how it maps to the original record.

    What a citation includes¶

    • Source system (e.g., Stripe, QuickBooks)
    • Record identifier (e.g., invoice ID, report name, transaction reference)
    • Query timestamp (when the data was fetched)
    • Direct link back to the source record in the native application

    This means no black-box answers. If the system says "Q2 subscription revenue was $847,230," you can click through and verify that figure in Stripe yourself.

    Dashboard Switching vs. AI Workspace Search¶

    The traditional workflow forces teams into a constant cycle of context-switching. Here's how the two approaches compare:

    Workflow Step Dashboard Switching AI Workspace Search
    1. Define the question Same starting point Same starting point
    2. Pick a tool Guess which dashboard has the answer No guessing — ask once
    3. Navigate Log in, find the right report, set filters Skip entirely
    4. Extract answer Read charts, export data, copy/paste Answer delivered inline
    5. Cross-reference Log into a second tool, repeat steps 2–4 Automatically correlated across tools
    6. Verify Manually check source records Citations linked to every source record
    7. Share Screenshot, paste into Slack/email Share a permalink to the query result
    Time per question 5–15 minutes 5–15 seconds

    The Cost of Switching¶

    Research on context-switching in knowledge work shows that each switch costs 15–25 minutes in lost focus. For a team of five analysts asking four questions per day across three tools each, that's 25+ hours per week spent navigating dashboards instead of acting on insights.

    AI Workspace Search collapses those seven steps into one: ask, and receive.

    How It Works¶

    1. Connect your tools — Link Stripe, Shopify, QuickBooks, and any other platform through CorpusIQ's connector library (OAuth or API key, one-time setup).
    2. Ask a question — Type a natural-language query like "Which marketing channel drove the highest net revenue last month?"
    3. Parallel query execution — CorpusIQ dispatches the question to every relevant connector simultaneously.
    4. AI correlation — The response engine merges, deduplicates, and correlates results across tools.
    5. Sourced answer — You receive a concise answer with citations and links back to source records.

    Frequently Asked Questions¶

    How many tools can I search at once?¶

    All of them. AI Workspace Search queries every connected tool in your workspace simultaneously. There's no limit on the number of connectors per query.

    What types of questions can I ask?¶

    Revenue analysis, customer segmentation, marketing attribution, churn forecasting, expense breakdowns, anomaly detection — any question that can be answered by data in your connected tools.

    Does it require SQL or coding?¶

    No. You ask questions in plain English. The AI translates your question into the appropriate queries for each underlying system.

    How are source citations provided?¶

    Every data point in every answer includes a citation block identifying the source system, record ID, query timestamp, and a direct link back to the original record.

    Is data real-time?¶

    Query freshness depends on the source system. Most connectors support near-real-time API access. Data warehouse connectors reflect your warehouse's refresh cadence.

    Can I save and share queries?¶

    Yes. Every query result has a permalink you can share. Team members can click the link and see the same answer with live-refreshed data.

    How does this compare to a BI tool like Looker or Tableau?¶

    BI tools require modeled data layers, SQL expertise, and dashboard-building. AI Workspace Search is designed for ad hoc questions — ask anything, get an answer immediately, no modeling required. It complements BI tools rather than replacing them.

    Which connectors are available?¶

    37+ native connectors covering payments, e-commerce, accounting, CRM, marketing, analytics, and data warehouses. See the full list in the Integrations documentation.
