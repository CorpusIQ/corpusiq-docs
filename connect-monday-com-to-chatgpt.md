# Connect Monday.com to ChatGPT

Query your Monday.com boards and projects directly from ChatGPT using CorpusIQ.

## Setup

1. [Connect Monday.com](https://corpusiq.io) in your CorpusIQ dashboard
2. Add CorpusIQ to ChatGPT as an MCP server: `https://mcp2.corpusiq.io/mcp`
3. Start asking questions

## Example Questions

- "What tasks are overdue across all my boards?"
- "Show me all items assigned to me with high priority"
- "Which projects have been in progress the longest?"
- "Summarize this week's completed items"

## What You Can Access

- Boards, groups, and items
- Column values and statuses
- Assignments and timelines
- Updates and activity logs

## Security

CorpusIQ uses read-only access. ChatGPT can query your Monday.com data but cannot create, update, or delete items.

See all [connectors](connectors.md) for other tools you can connect.
