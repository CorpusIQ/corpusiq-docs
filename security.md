# Security & Compliance

CorpusIQ is designed for business data security. Your data never leaves your tools — we query it live and pass results directly to your AI.

## Key Security Features

- **Read-only access** — Minimum permissions on every connector
- **No data warehouse** — Nothing is copied or stored
- **Per-user authentication** — Each connection is scoped to the authenticating user
- **Source citation** — Every answer shows where the data came from

## Compliance

CorpusIQ is SOC 2 Type II compliant (in progress). Enterprise customers can request:

- Custom data retention policies
- Audit logging
- SSO/SAML integration
- Private deployment options
