# Connect Gmail to ChatGPT

Query your Gmail inbox directly from ChatGPT using CorpusIQ.

## Setup

1. [Connect Gmail](https://corpusiq.io) in your CorpusIQ dashboard
2. Add CorpusIQ to ChatGPT as an MCP server: `https://mcp2.corpusiq.io/mcp`
3. Start asking questions

## Example Questions

- "Summarize unread emails from this week"
- "Find all emails from example.com about the contract"
- "What meetings do I have tomorrow?"
- "Show me emails with attachments I haven't downloaded yet"

## What You Can Access

- Emails (subject, sender, date, body)
- Attachments (names and types)
- Calendar events (via Google Workspace connector)
- Search across all labels and folders

## Security

CorpusIQ uses read-only access. ChatGPT can query your Gmail but cannot send, delete, or modify emails.

See all [connectors](connectors.md) for other tools you can connect.
