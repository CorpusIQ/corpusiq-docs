# AI for Retail — POS, Inventory, Multi-Channel Analytics

Retailers operate across channels that don't talk to each other: POS systems in physical stores, ecommerce platforms online, marketplace channels, inventory databases, and marketing platforms — each with its own dashboard and reporting format. **AI for retail** through CorpusIQ MCP collapses this fragmentation: connect your POS, inventory, ecommerce, and marketing tools to ChatGPT or Claude, and get unified, cross-channel answers in plain English — instantly, from live data.

No CSV exports. No dashboard hopping. No reconciling numbers across tabs. One question, one answer, your entire retail operation.

## Retail-Specific KPIs at Your Fingertips

| KPI | Formula | Ask ChatGPT |
|---|---|---|
| **Sales per Square Foot** | Revenue ÷ Selling Area | "Sales per square foot by store this quarter?" |
| **Inventory Turnover** | COGS ÷ Average Inventory | "Inventory turns by category over the last 12 months?" |
| **Sell-Through Rate** | Units Sold ÷ Units Received | "Sell-through rate by product line this season?" |
| **GMROI** | Gross Margin ÷ Average Inventory Cost | "GMROI by department?" |
| **Stock-to-Sales Ratio** | Inventory Value ÷ Sales | "Stock-to-sales ratio by category this month?" |
| **Markdown Rate** | Markdown Value ÷ Total Sales | "Markdown percentage by store vs. online?" |
| **Conversion Rate** | Transactions ÷ Foot Traffic | "Conversion rate trend by store over the last 90 days?" |

No SQL. No dashboard configuration. Just ask the question.

## Security: Read-Only Across Retail Systems

| System | Connection Method | Permissions |
|---|---|---|
| **POS Database** | Connection string (PostgreSQL/MSSQL/MongoDB) | SELECT-only — read sales, inventory, customers |
| **Shopify** | OAuth 2.0 | Read products, orders, customers, analytics |
| **Amazon Seller Central** | Read-only API | Read orders, inventory, financials |
| **eBay** | Seller API (read scopes) | Read orders, listings, analytics |
| **Google Analytics** | OAuth 2.0 | Read traffic, conversions, ecommerce data |

No retail data is stored, cached, or retained after a query. ChatGPT cannot modify inventory, change prices, process refunds, or alter any retail data.
