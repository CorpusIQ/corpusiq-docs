# AI voor Nederlandse Bedrijven — Veilig Verbonden met Al je Tools

Nederlandse bedrijven gebruiken een combinatie van tools: boekhouding, CRM, e-commerce, marketing. Tot nu toe betekende een geïntegreerd overzicht: CSV-exports, eindeloze spreadsheets en verloren uren. **AI voor Nederlandse bedrijven** via CorpusIQ MCP verandert dat: sluit je tools één keer aan, stel vragen in het Nederlands aan ChatGPT of Claude, en krijg live antwoorden met bronvermelding — zonder dat er gegevens worden verplaatst of opgeslagen.

Geen exports meer. Geen SQL. Geen spreadsheets. Gewoon antwoorden, in het Nederlands, op basis van je eigen data.

## AVG-Compliance door Architectuur

CorpusIQ voldoet aan de AVG-vereisten door het ontwerp:

| AVG-Principe | CorpusIQ Implementatie |
|---|---|
| **Dataminimalisatie** | Geen gegevensopslag — queries zijn vluchtig |
| **Doelbinding** | Alleen-lezen OAuth-toegang — alleen om je vraag te beantwoorden |
| **Beveiliging** | TLS 1.3-versleuteling, OAuth 2.0-authenticatie, tokenrotatie |
| **Toegangscontrole** | Erft je bestaande gebruikersrechten van elke aangesloten tool |
| **Geen AI-training** | Je gegevens worden nooit gebruikt om AI-modellen te trainen |
