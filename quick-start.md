# Quick Start

Get CorpusIQ connected to your AI tools in under 5 minutes.

## 1. Sign Up

Go to [corpusiq.io](https://corpusiq.io) and create an account. Free 30-day trial, no credit card required.

## 2. Connect Your First Data Source

From the dashboard, click **Add Connector** and choose your first data source. We support 37+ tools:

- Stripe for payments and subscriptions
- Shopify for ecommerce data
- QuickBooks for accounting
- Google Analytics for web traffic
- HubSpot for CRM
- And 32+ more

Each connector needs a one-time authentication. Click Connect, approve the read-only access, and you're done. Takes about 30 seconds per connector.

## 3. Connect Your AI Tool

CorpusIQ works with any MCP-compatible AI platform. The most common setups:

### ChatGPT
Add CorpusIQ as an MCP server in ChatGPT settings. Full guide: [ChatGPT Integration](chatgpt-integration.md)

### Claude
Add to Claude Desktop's MCP config. Full guide: [Claude Integration](connectors/ai-mcp-server-for-business-data.md)

### Perplexity
Add as a data source in Perplexity settings. Full guide: [Perplexity Integration](connectors/perplexity-business-data-mcp.md)

For any other MCP-compatible tool, use this server URL:

```
https://mcp2.corpusiq.io/mcp
```

## 4. Ask Real Questions

Once connected, you can ask questions that pull live data from your tools:

- "What was my ROAS across Meta and Google Ads last month?"
- "Show me Stripe revenue vs QuickBooks expenses for Q2"
- "Which Shopify products had the highest return rate?"
- "Compare HubSpot deal velocity this quarter vs last quarter"

CorpusIQ runs cross-source queries automatically. You get answers with cited data, not summaries.
