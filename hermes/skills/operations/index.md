# Operations Skills

Operations skills handle the infrastructure of running autonomous agents: email monitoring, cron management, system audits, communication priority, and job application automation.

## Email Operations
- Dual-account Gmail monitoring (media at, info at)
- OAuth token lifecycle management
- Professional HTML email response standards
- Lead classification and routing by business domain
- Inbound response personalization with prospect research

## System Management
- Cron delivery audit — verify all crons actually deliver by running once
- System discovery audit — find all installed tools and their states
- Health monitoring and anomaly detection
- Browser/web extraction setup and configuration
- Local AI infrastructure management

## Communication Discipline
- Communication priority and SLA hierarchy
- Execution-first — minimize explanation, maximize action
- Done-only reporting — never report scheduled, queued, or planned work
- Professional custom email templates for all outbound

## Autonomous Recruiting
- Job search and application automation
- Resume parsing and cover letter generation
- Application monitoring across platforms
- Email reply forwarding for job responses
- BuiltIn.com and multiple platform support

## Governance
- System registry for component tracking and deduplication
- Improvement recommendation engine
- Daily self-improvement cycle at 23:00
- Session database optimization for context management

All operations skills follow the principle: verify, do not trust. Cron output is verified by actual execution, not status flags. Email delivery is confirmed, not assumed. System health is measured, not estimated.
