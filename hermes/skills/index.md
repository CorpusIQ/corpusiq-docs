## Writing Custom Skills

Skills are stored in `~/.hermes/profiles/{profile}/skills/`. Structure:

```markdown
---
name: my-skill
description: What this skill does
---

# Skill Title

Step-by-step workflow with tool calls, pitfalls, and verification.
```
