## Key Capabilities

### Platform Selection Framework
Decision matrix matching your ICP (Ideal Customer Profile) to the right platforms:
- **LinkedIn**: B2B, professionals, thought leadership
- **Twitter/X**: Indie hackers, tech, real-time discussion
- **Instagram**: Visual products, lifestyle brands (B2C)
- **TikTok**: Younger demographics, viral content
- **YouTube**: Educational long-form, authority building
- **Reddit**: Niche communities, problem-solving

Rule: Start with ONE platform. Master it before expanding.

### Content Mix (80/20 Rule)
- **80% value**: Educational, inspirational, personal, engagement-driving
- **20% promotion**: Product mentions, offers, CTAs

### Platform-Specific Writing Structures
Detailed templates for each platform:
- LinkedIn: Hook → Body (3-5 paragraphs) → CTA with line breaks
- Twitter/X: Thread structure (Hook → 8 breakdown tweets → CTA)
- Instagram: Visual-first with hook caption, value paragraphs, CTA

### Posting Frequency Guide
- LinkedIn: 3-5x/week
- Twitter: 5-10x/week (or daily threads)
- Instagram: 3-5x/week + daily stories
- TikTok: Daily (5x/week minimum)

### Goal-to-Content Mapping
Brand awareness → shareable content, hooks. Community → replies, conversations. Lead gen → clear CTAs, lead magnets. Authority → educational, case studies, original thinking.

### Example: LinkedIn Post Structure

```
HOOK (first 1-2 lines — must grab attention in the feed)
BODY (story, insight, or framework — 3-5 short paragraphs with line breaks)
CTA (ask a question, invite comments, or link to something)
```

### Example: Twitter Thread Structure

```
1/🧵 [Hook - the main insight]
2/ [Supporting point 1]
3/ [Supporting point 2]
4/ [Example or evidence]
5/ [Conclusion + CTA]
```
