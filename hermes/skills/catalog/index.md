## Using Skills

Native skills load automatically when the agent profile matches the skill trigger. Marketplace skills install once:

```bash
# Install a skill
npx skills add coreyhaines31/marketingskills@seo-audit

# Install a full pack
npx skills add coreyhaines31/marketingskills

# List installed skills
hermes skills list
```

## Adding New Skills

Skills are added through:
1. **CorpusIQ team** — new native skills for growth, ops, or video
2. **skills.sh** — community marketplace skills
3. **Manual creation** — `skill_manage(action='create')` in Hermes

This catalog should be updated when new skills are added.
