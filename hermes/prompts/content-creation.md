# Content Creation Prompts  --  Blog, Social, Email & SEO Templates

---

## Blog Posts and Articles

### Long-Form Article

```
Write a [WORD COUNT]-word blog post about [TOPIC].

Target audience: [DESCRIBE AUDIENCE  --  e.g., "engineering managers at mid-size SaaS companies"]
Tone: [e.g., authoritative but conversational, technical but approachable]
Goal: [e.g., educate readers about X so they try Y]

Structure:
1. Compelling headline with a numbered or how-to angle
2. Hook paragraph that names the reader's pain point
3. [3-5] main sections with clear subheadings
4. Concrete examples or case studies in each section
5. Actionable takeaways summary
6. Call-to-action: [DESCRIBE CTA]

Include internal linking opportunities to [LIST RELATED POSTS] and suggest 2-3 places to naturally insert them.

SEO: Primary keyword "[KEYWORD PHRASE]" with density ~1.5%. Secondary keywords: [LIST].
```

### Listicle / Roundup Post

```
Create a listicle: "[NUMBER] [ADJECTIVE] [THINGS] for [AUDIENCE] in [YEAR]"

For each item include:
- Bold headline (50-70 characters)
- 2-3 paragraph explanation of what it is and why it matters
- A concrete example or statistic
- "Who it's best for" or "When to use it" callout

Introduction: Hook with the problem these tools/solutions solve.
Conclusion: Decision framework for choosing between options.

Include a comparison table at the end summarizing key differences.
```

## Social Media Content

### Multi-Platform Post Set

```
Generate a coordinated set of social media posts for [ANNOUNCEMENT/CONTENT PIECE].

Source material: [LINK OR PASTE KEY POINTS]

Create variations for:
1. LinkedIn (1300 characters max)  --  professional tone, 3-5 hashtags
2. Twitter/X (280 characters)  --  punchy, 1-2 hashtags
3. Facebook (ideal 80-120 characters)  --  conversational, emoji-friendly
4. Instagram caption (2200 characters max)  --  visual-first description, 5-10 hashtags

Each post should highlight a different angle of the same content so followers on multiple platforms don't see identical copy.

Best posting times: [SPECIFY TIMEZONE AND PREFERRED WINDOWS]
```

### Engagement Prompts

```
Generate [N] social media engagement prompts for [PLATFORM] about [TOPIC].

Each prompt should:
- Ask a question that invites genuine response
- Include a poll option or "A or B" choice where applicable
- Be something the target audience of [AUDIENCE] would care enough to answer
- Avoid generic engagement bait ("double tap if you agree")

Format each as: the post text, then the intended engagement mechanism (poll, open question, fill-in-blank).
```

## Email Campaigns

### Nurture Sequence

```
Design a [N]-email nurture sequence for [PURPOSE  --  e.g., onboarding, re-engagement, product education].

Subscriber context: [WHAT THEY SIGNED UP FOR / WHERE THEY ARE IN THE JOURNEY]
Goal: [DESIRED ACTION AFTER THE SEQUENCE]
Brand voice: [DESCRIBE  --  e.g., "friendly, jargon-free, slightly playful"]

For each email provide:
1. Subject line (with A/B test variant)
2. Preview text
3. Body: opening hook, core message (2-3 paragraphs), CTA
4. Send timing: Day [X] at [TIME], with rationale

Include a decision branch: if they [DO/DON'T] take the action, what happens next?
```

### Newsletter Template

```
Create a reusable email newsletter template for [BRAND/COMPANY].

Sections (in order):
1. Personal note from [SENDER ROLE] (200-400 characters)
2. Featured content block (headline + 2-sentence teaser + link)
3. "What we're reading"  --  3 curated links with one-line commentary
4. Community spotlight or customer story (if applicable)
5. Clear unsubscribe footer with brand personality

Constraints: Must render well in Gmail, Outlook, and Apple Mail. Use inline CSS. Total width 600px. Keep under 15KB to avoid Gmail clipping.
```

## Video Scripts

### Tutorial / How-To

```
Write a script for a [DURATION]-minute tutorial video on [TOPIC].

Format: Two-column script (Visual | Audio)

Sections:
- 0:00-0:30 Hook: Show the result first, then state the problem
- 0:30-[X]: Step-by-step walkthrough with screen share directions
- [X]-[END]: Recap + call-to-action

Pacing: [FAST/MEDIUM/DETAILED]. Include B-roll suggestions and text overlay timings.
```

## SEO Content

### Content Brief

```
Create an SEO content brief for a page targeting the keyword "[PRIMARY KEYWORD]".

Target audience search intent: [INFORMATIONAL/COMMERCIAL/TRANSACTIONAL/NAVIGATIONAL]

Include:
1. Recommended title tag (50-60 characters) and meta description (150-160 characters)
2. H1 and 5-8 H2/H3 subheadings mapped to related queries
3. Competing pages to analyze: [LIST URLS]
4. Content length recommendation with rationale
5. Internal link targets and suggested anchor text
6. Image/visual recommendations
7. Schema markup type to use

The brief should be complete enough to hand to a writer unfamiliar with the topic.
```
