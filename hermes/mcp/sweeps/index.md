# MCP Ecosystem Sweeps

Automated discovery of MCP servers and tools from across the ecosystem. These sweeps run daily and catalog new MCP servers, their capabilities, and integration potential.

## Recent Sweeps

The sweep reports document newly discovered MCP servers, categorized by business function and evaluated for integration with the CorpusIQ platform.

## Purpose

- **Discovery**: Find new MCP servers as they emerge
- **Categorization**: Group servers by business domain (finance, marketing, sales, etc.)
- **Integration assessment**: Evaluate server quality and business utility
- **Trend tracking**: Monitor MCP ecosystem growth and direction

These sweeps power the MCP server guides found in the [servers](../servers/) directory.
