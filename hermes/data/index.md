# Ecosystem Data

Data files powering the Hermes ecosystem discovery engine — approved repositories, pending reviews, contributor tracking, and inclusion criteria.

## Data Files

- **Inclusion Criteria** (`inclusion_criteria.md`) — Standards for ecosystem listing
- **Approved Repos** (`approved_repos.json`) — Repositories accepted into the ecosystem
- **Pending Review** (`pending_repos.json`) — Repositories awaiting evaluation
- **Contributors** (`contributors.json`) — Community contributor tracking
- **Review Log** (`review_log.json`) — Audit trail of review decisions
- **Latest Discovery Report** (`latest_discovery_report.json`) — Most recent scan results
