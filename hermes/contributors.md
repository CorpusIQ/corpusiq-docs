## How to Contribute

1. Find a Hermes-related repo not yet in the directory
2. [Open an issue](https://github.com/CorpusIQ/corpusiq-docs/issues/new?labels=pending-review) with the GitHub URL
3. The discovery engine scores it automatically
4. Repos scoring ≥ 70 are auto-approved and added to the ecosystem
