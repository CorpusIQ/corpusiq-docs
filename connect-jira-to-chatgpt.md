# Connect Jira to ChatGPT

Query your Jira projects and issues directly from ChatGPT using CorpusIQ.

## Setup

1. [Connect Jira](https://corpusiq.io) in your CorpusIQ dashboard
2. Add CorpusIQ to ChatGPT as an MCP server: `https://mcp2.corpusiq.io/mcp`
3. Start asking questions

## Example Questions

- "What bugs are open and assigned to the engineering team?"
- "Show me all issues in the current sprint sorted by priority"
- "How many tickets were resolved last week?"
- "Which epics are at risk of missing the deadline?"

## What You Can Access

- Projects, issues, and epics
- Sprints and backlogs
- Assignments and statuses
- Comments and work logs

## Security

CorpusIQ uses read-only access. ChatGPT can query your Jira data but cannot create, update, or transition issues.

See all [connectors](connectors.md) for other tools you can connect.
