# Reporting

CorpusIQ lets you ask questions instead of building reports. But when you need to share insights, here's how.

## AI-Generated Reports

Every query result can become a report. Ask your AI tool to format the answer as:

- Executive summaries
- Weekly performance snapshots
- Board-ready financial overviews
- Campaign performance breakdowns

## Scheduled Queries

For recurring reports, set up scheduled queries in the dashboard:

1. Write your question (e.g., "What was this week's revenue across all channels?")
2. Set the schedule (daily, weekly, monthly)
3. Choose delivery (email, Slack, or webhook)

## Dashboard Exports

CorpusIQ doesn't have a traditional dashboard — your AI tool is the interface. But you can:

- Export query results as CSV
- Pipe results to Google Sheets
- Trigger reports via API
