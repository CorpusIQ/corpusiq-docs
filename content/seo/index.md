# SEO Content Library

Targeted landing pages and content optimized for search engines, covering MCP integration with specific business tools and platforms.

## Structure

The SEO library contains focused, keyword-optimized pages for high-intent searches related to MCP, AI agents, and business tool connectivity. Each page targets a specific search query pattern and provides actionable setup instructions.

## Categories

- **Platform-specific guides**: Connect [tool] to [AI platform]
- **Industry use cases**: MCP for [industry]
- **Technical deep dives**: How MCP works, security, architecture
